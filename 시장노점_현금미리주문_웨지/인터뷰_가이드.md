# 상인 인터뷰 가이드 (5~7곳) — 웨지 검증용

> 대상: 전통시장 내 **단골 미리주문 적합 업종**(반찬·떡/방앗간·정육/생선·족발보쌈전·김밥도시락). 노점은 "단골 예약형"만.
> 목적: 3대 게이트(행동·WTP·합법카피) 검증. 한 곳당 10~15분, 라포 먼저.
> 원칙: 유도질문 금지. "탈세"·"세금" 먼저 꺼내지 말 것(상인이 꺼내면 받기만). 실제 행동/숫자를 캐물을 것.

## A. 현황 (라포 + 배경)
1. 하루 주문 중 **전화로 미리 주문**받는 게 몇 건쯤 되나요? 주로 어떤 손님(단골/신규)인가요?
2. 미리 주문 받을 때 불편한 점은? (받아적기·통화중·시간 약속·누락 등)
3. **카드단말기 쓰시나요?** 손님 결제는 현금/카드 비율이 어느 정도?

## B. 경쟁 대안 (왜 안/못 쓰나 — 핵심)
4. **네이버 주문이나 카카오 주문하기** 써보셨거나 들어보셨어요? (안 쓰면) 왜 안 쓰세요?
5. (해당 시) **영업신고증/사업자등록** 때문에 가입이 번거롭거나 막힌 적 있나요?
6. 그런 데는 **손님이 미리 카드로 결제**해야 하는데, 그게 장사에 맞나요/불편한가요?

## C. 행동 검증 (GO의 핵심)
7. 만약 **손님이 카톡으로 미리 주문 → 와서 현금/카드로 직접 결제**하는 페이지가 있으면, 단골들이 실제로 쓸 것 같으세요? 누가 쓸까요?
8. 손님이 미리 주문하면 사장님 장사에 **뭐가 가장 좋을까요?**(대기 줄임·미리 준비·매출↑ 중)
9. 반대로, 미리주문이 **안 맞는 메뉴/상황**은 뭐가 있을까요? (행동 부적합 신호 포착)

## D. WTP (지불의향)
10. 이런 도구가 **월 얼마면** 쓰시겠어요? (앵커 없이 먼저 물은 뒤) 5천 / 9천 / 1만5천 중 어디까지?
11. **정부·시장 지원금으로 자부담이 거의 0**이면 바로 쓰시겠어요? (스마트상점 바우처 가정)
12. 지금 장사에 **매달 돈 내고 쓰는 디지털 도구**(POS·문자·광고 등) 있나요? 얼마요?

## E. 합법·카피 (탈세 프레이밍 회피 확인)
13. 주문 내역이 **기록으로 남는 것**에 대해 어떻게 생각하세요?(거부감 강하면 합법코어와 충돌 신호 — `04` 참고)
14. (상인이 세금 얘기 꺼내면만) 어떤 점이 걱정이세요? — 듣기만, 탈세 조장 약속 금지.

## F. 마무리
15. 주변에 이거 쓸 만한 **다른 상인** 있을까요? (시장단위 온보딩 단서) 상인회는 활발한가요?

---
### 기록 시트(곳당)
- 업종 / 영업신고증 유무 / 카드단말기 / 현금비율 / 전화미리주문 건수
- 네이버·카카오 안 쓰는 이유 / 미리주문 행동 예상(쓴다·안쓴다·조건부) / 적합도
- WTP(단독) / WTP(바우처) / 기존 디지털 지출 / 기록 거부감 / 소개 가능 상인 수

### 게이트 판독
- **행동**: 7·8·9에서 "단골이 쓴다"가 다수 → 통과 / "충동구매라 안 쓴다"가 다수 → 보류.
- **WTP**: 11(바우처 자부담)에 다수 긍정 → 통과 / 0원만 → 모델 재설계.
- **합법**: 13에 기록 거부감 강함 + 14가 노골 탈세 → 위험(타겟·카피 재고).
